# Kubernetes Deployment Lab

A hands-on DevOps learning project covering:

- HTML/CSS/JavaScript application development
- Docker containerization
- GitHub Actions CI/CD
- Self-hosted GitHub runner
- Kubernetes deployment
- NGINX Ingress
- TLS
- Rolling Updates
- Blue-Green Deployment
- Canary Deployment
- Monitoring

## Architecture

Developer

↓

GitHub Repository

↓

GitHub Actions

↓

Docker Image

↓

Kubernetes Cluster

↓

NGINX Ingress

↓

Application

## Learning Goal

Understand the complete application lifecycle from code commit to Kubernetes deployment.
